# Buat Laporan di Sini

Pada program assignment 0 ini, diperintahkan untuk membuat sebuah program yang menerima masukan user dan mencetak masukan yang dimasukkan user. Di kode ini terdapat fitur yang membuat binary tidak langsung tertutup ketika user menekan tombol Enter setelah memasukkan masukan. Library yang digunakan di program ini adalah iostream.
